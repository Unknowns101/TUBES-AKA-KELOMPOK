# TUBES-AKA-KELOMPOK

### Penjelasan Singkat Mengenai Program :

Program ini digunakan untuk membandingkan dua metode dalam menghitung jumlah digit suatu bilangan, yaitu metode iteratif dan rekursif. Melalui aplikasi Streamlit, pengguna dapat memasukkan bilangan bulat, kemudian program akan menghitung hasil penjumlahan digit menggunakan kedua metode sekaligus, mengukur waktu eksekusi masing-masing, dan menampilkan grafik perbandingan performanya. Program juga menampilkan penjelasan singkat mengenai kompleksitas algoritma, serta menangani kemungkinan terjadinya stack overflow pada metode rekursif ketika bilangan yang dimasukkan terlalu panjang.
